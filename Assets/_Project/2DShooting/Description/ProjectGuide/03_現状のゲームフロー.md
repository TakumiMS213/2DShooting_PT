# 03 現状のゲームフロー

## 画面の流れ

```text
TitleScene -> GameScene -> ResultScene
```

## TitleScene

タイトル画面です。

STARTボタン、`Enter`、`Space` でゲーム画面へ進みます。

## GameScene

実際に遊ぶ画面です。

流れ:

1. Playerを動かす。
2. 弾を撃つ。
3. 弾をEnemyに当てる。
4. EnemyのHPが減る。
5. Enemyを倒す。
6. ResultSceneへ進む。

## ResultScene

結果画面です。

- Retry: もう一度ゲーム画面へ進む。
- Title: タイトル画面へ戻る。

キー操作:

- `Enter` または `Space`: Retry
- `Esc`: Title
