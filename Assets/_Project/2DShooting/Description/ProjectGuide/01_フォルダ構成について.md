# 01 フォルダ構成について

## よく使うフォルダ

```text
Assets/_Project/2DShooting
├── Configs
├── Description
├── Imagea
├── Prefabs
├── Scenes
└── Scripts
```

## Configs

ゲームの調整ファイルです。

初心者が一番よく触る場所です。

- `PlayerConfig.asset`
- `EnemyConfig.asset`
- `BulletConfig.asset`
- `StageConfig.asset`

## Scenes

ゲームの画面です。

- `TitleScene.unity`
- `GameScene.unity`
- `ResultScene.unity`

## Prefabs

ゲームに出る部品です。

- `Player.prefab`
- `Enemy.prefab`
- `Bullet.prefab`

## Imagea

仮画像の置き場です。

- `PrototypeSquare.png`

## Description

この説明資料の置き場です。

## Scripts

ゲームを動かす命令の置き場です。

最初は `Scripts` より先に、`Scenes` と `Configs` を見てください。
