# 04 改造のヒント

## 最初はConfigsだけ触る

最初の改造では、スクリプトを書き換えずに `Configs` の値だけを変えます。

値を変えたら、Playして違いを確認してください。

## PlayerConfig

```text
Assets/_Project/2DShooting/Configs/PlayerConfig.asset
```

おすすめ変更:

- `Move Speed`: 自機の移動速度。
- `Tint`: 自機の色。
- `Local Scale`: 自機の大きさ。
- `Fire Interval`: 弾を撃つ間隔。
- `Movement Min / Max`: 動ける範囲。

## EnemyConfig

```text
Assets/_Project/2DShooting/Configs/EnemyConfig.asset
```

おすすめ変更:

- `Max Health`: 敵のHP。
- `Enemy Count`: 敵の数。
- `Move Speed`: 敵の動く速さ。
- `Patrol Distance`: 敵が左右に動く幅。
- `Tint`: 敵の色。

## BulletConfig

```text
Assets/_Project/2DShooting/Configs/BulletConfig.asset
```

おすすめ変更:

- `Damage`: 弾のダメージ。
- `Speed`: 弾の速さ。
- `Pool Size`: 同時に使える弾の数。
- `Life Time`: 弾が消えるまでの時間。
- `Trajectory`: 弾道。

## StageConfig

```text
Assets/_Project/2DShooting/Configs/StageConfig.asset
```

おすすめ変更:

- `Required Enemy Defeats`: クリアに必要な撃破数。
- `Result Delay Seconds`: クリア後に結果画面へ進むまでの待ち時間。
- `Background Scroll Speed`: 背景の流れる速さ。

## 複数の敵を出す時の注意

`EnemyConfig.asset` の `Enemy Count` を増やしたら、`StageConfig.asset` の `Required Enemy Defeats` も同じ数にしてください。

例:

- `Enemy Count = 3`
- `Required Enemy Defeats = 3`
