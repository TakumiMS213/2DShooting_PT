# 00 概要

## この資料の目的

この資料は、初心者が実際に触る場所だけを説明します。

細かい設計や難しい用語は扱いません。

## このゲームでできること

- タイトル画面からゲームを始める。
- 自機を動かす。
- 弾を撃つ。
- 敵に弾を当てる。
- 敵を倒したらリザルト画面へ進む。

## 主に触る場所

- `Scenes`: 画面を見る。
- `Configs`: 数値や見た目を変える。
- `Prefabs`: Player、Enemy、Bulletの部品を見る。
- `Imagea`: 仮画像を見る。

## まずやること

1. `Scenes/TitleScene.unity` を開く。
2. Playする。
3. ゲームの流れを確認する。
4. `Configs` の値を変えて、動きが変わることを確認する。
