# スクリプト案内

ゲーム本体は `Scripts` 直下の5ファイルだけで動きます。

| ファイル | 役割 | 試せること |
| --- | --- | --- |
| `GameRules.cs` | 敵を出し、撃破数を数える | 敵の出し方、クリア条件 |
| `Player.cs` | 入力、移動、ショット | 操作方法、移動方法 |
| `Bullet.cs` | 弾の移動と衝突 | 新しい弾道 |
| `Enemy.cs` | 敵の体力と移動 | 新しい敵の動き |
| `ScreenFlow.cs` | 3画面の切り替え | 画面を増やす |

`Editor/PrototypeSetup.cs` は教材を再生成するためのEditor専用スクリプトです。ゲーム実行中には使われません。
