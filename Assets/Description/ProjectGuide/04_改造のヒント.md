# 試してみるヒント

一度にたくさん変えず、**ひとつ変える → Playする → 違いを言葉にする**の順で試してください。

## まずは数字を変える

- Playerの `Move Speed` を `1` から `10` にする
- Playerの `Shot Interval` を `1` から `0.5` にする
- GameRulesの `Enemies To Clear` を `4` から `8` にする
- Enemyの `Health` を `3` から `1` にする

## 次に形を変える

Prefabをダブルクリックして、TransformのScaleやSprite RendererのColorを変えます。

- Playerを横長や縦長にする
- Enemyの色を変える
- Bulletを細長くする

見た目は1つの四角形です。まずはScaleとColorだけで違いを試せます。

## 慣れたらルールを変える

- Bulletの `Path` を `Wave` にする
- GameRulesの `Spawn Step` を変えて敵の並びを作る
- GameRulesの `Spawn Interval` を `1` にして、敵を1秒おきに出す
- 2種類のEnemy Prefabの順番を入れ替える

## Sceneビューの設計線

- Playerを選択すると、水色の枠が移動可能範囲を示します。
- 黄色い丸が弾の発射位置です。
- GameRulesを選択すると、オレンジの印が敵の出現位置を示します。

数値と結果の間を、目で追えるようにしています。
