# フォルダ構成

```text
Assets
├── Description   読み物
├── Images        仮画像
├── Prefabs       Player / Enemy / Bulletの部品
├── Scenes        Title / Game / Resultの3画面
└── Scripts
    ├── *.cs      ゲーム本体（ここだけ読めばOK）
    └── Editor    教材の再生成ツール
```

最初は `GameScene` と `Prefabs` を触ります。スクリプトで調整してみたくなったら `Scripts` を開いてください。

Unityの描画に必要なファイルは `Settings` にまとまっています。通常は触らなくて構いません。
