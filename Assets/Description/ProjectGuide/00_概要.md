# P班特訓 ― 2Dシューティング編

このプロジェクトは、Unityに慣れていない新入生が、ゲームを遊びながら少しずつ仕組みに触れるための教材です。

## 最初の3分

1. `Scenes/TitleScene` を開いてPlayします。
2. ゲームを一度クリアします。
3. `Scenes/GameScene` を開きます。
4. Hierarchyの `★ GameRules — 敵の数や並びを変える` を選びます。
5. `Enemies To Clear` を変えて、もう一度Playします。

これだけで最初の調整は完了です。分からないところは、周りの人と相談しながら進めてください。

## 触る場所

- `GameScene / ★ GameRules`: 敵の数、出現位置、クリア条件
- `GameScene / Player`: 移動速度、連射間隔、移動範囲
- `Prefabs/Bullet`: 弾の速さ、威力、軌道
- `Prefabs/Enemy_Scout` と `Enemy_Heavy`: 敵の体力と動き
- 各Sceneの `UI_...`: 画面の文字、色、配置

UIはScreen Space - Cameraです。Sceneビューでカメラの近くに表示されるため、巨大なCanvasを探す必要はありません。

## 困ったとき

メニューの `Tools > 2D Shooting > 教材を最初の状態に戻す` で、この教材の初期状態を再生成できます。
